# Cloudflare Pages Deploy Folder

This directory is the clean publish root for the volleyball test site.

Upload this directory as the repository root for GitHub + Cloudflare Pages.

Contents:

- `index.html`
- `image/`

Cloudflare Pages settings:

- Framework preset: `None`
- Build command: leave empty
- Build output directory: `/`
- Root directory: leave empty if this folder is the repo root

No build step is required. This is a static site.
